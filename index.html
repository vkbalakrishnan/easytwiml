Generate TwiML's on the fly.

Options

  • text : (Text) What the call should say. eg:http://vkb.me/easytwiml/xml/?text=welcome%20to%20twilio
  • voice : By default "Man" voice is used. Otherwise set voice as "Woman". Should be used with 'text' parameter. eg:http://vkb.me/easytwiml/xml/?text=welcome%20to%20twilio&voice=woman
  • json : For complex TwiMLs.

JSON Example

eg:http://vkb.me/easytwiml/xml/?json={"Say":"welcome to twilio","Record":{"@":{"maxLength":20}}}

For more details on json formatting refer JS2XMLParser Github Page
